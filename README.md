# Suzerain Data Viewer

Suzerain Data Viewer is a tool that converts JSON data generated by [Suzerain Data Dumper](https://github.com/suzerain-modding/suzerain-data-dumper) into a static HTML site for easy inspection.

## Installation and Usage

Suzerain Data Viewer has builds for Windows x64 and Linux x64 (glibc).

1. Download the latest release from [releases](https://github.com/suzerain-modding/suzerain-data-viewer/releases). Place it anywhere on your computer.
2. Use [Suzerain Data Dumper](https://github.com/suzerain-modding/suzerain-data-dumper) to dump the game data.
3. Open terminal in the directory containing the files generated by Suzerain Data Dumper, or open terminal and `cd` to that directory.
4. Run the executable.
   - If on Windows, just copy and paste the path to the executable in the terminal to run it.
5. Select the JSON file to read.
6. Once done, open the generated `out/index.html` file to inspect.
