# Suzerain Data Viewer

Suzerain Data Viewer is a tool that converts JSON data generated by [Suzerain Data Dumper](https://github.com/suzerain-modding/suzerain-data-dumper) into a static HTML site for easy inspection.

**Note:** Some parts of this project were created with AI assistance.
